# 089: 「番人は属性の入口に立つ」のヒント

[問題](../problems/089-descriptor-guard.md) / [解答](../solutions/089-descriptor-guard.md)

**難易度:** ☆☆☆☆☆

??? tip "ヒント1"

    デスクリプタは、クラス属性として置かれたオブジェクトです。
    `__get__` や `__set__` を持つと、属性アクセスや代入に介入できます。

??? tip "ヒント2"

    `__set_name__(owner, name)` は、デスクリプタがどの属性名に代入されたかを受け取ります。
    この名前から、インスタンス側に保存するための別名を作れます。

??? tip "ヒント3"

    `__get__(self, instance, owner)` で `instance is None` のときは、クラスからアクセスされています。
    この場合はデスクリプタ自身を返すのが扱いやすいです。

??? tip "ヒント4"

    値はデスクリプタ自身に保存してはいけません。
    デスクリプタはクラスで共有されるため、インスタンスごとの値は `instance` 側に保存します。
