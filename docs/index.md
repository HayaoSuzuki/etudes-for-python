# Etudes for Python

このサイトは、プログラミングを小さな練習問題の連なりとして学ぶための演習書です。

構成は、数学の演習書にならって次の3部に分けます。

- [問題](problems/index.md): まず自力で考えるための本文です。
- [ヒント](hints/index.md): 解法を直接読まずに詰まりをほどくための補助です。
- [解答](solutions/index.md): 実装例、考え方、検証をまとめます。

各演習は `001-even-or-odd` のような共通IDを持ちます。
問題、ヒント、解答の各ページで同じIDを使うと、相互参照しやすくなります。
