# 009: 「リストの両端に王がいる」のヒント

[問題](../problems/009-min-max.md) / [解答](../solutions/009-min-max.md)

**難易度:** ☆☆

??? tip "ヒント1"

    最初の要素を、仮の最小値と最大値にできます。

??? tip "ヒント2"

    残りの要素を順に見て、より小さい値やより大きい値を見つけたら更新します。

??? tip "ヒント3"

    `numbers[0]` で最初の要素を取り出し、`numbers[1:]` で残りの要素を取り出せます。
