# 025: 「単語は文書番号を覚えている」のヒント

[問題](../problems/025-defaultdict-inverted-index.md) / [解答](../solutions/025-defaultdict-inverted-index.md)

**難易度:** ☆☆☆

??? tip "ヒント1"

    戻り値は、単語をキー、文書番号のリストを値にする辞書です。
    単語を見つけるたびに、その単語のリストへ文書番号を追加します。

??? tip "ヒント2"

    まだ出ていない単語でも空リストから始められるように、`defaultdict(list)` が使えます。

??? tip "ヒント3"

    1つの文書に同じ単語が複数回出ても、文書番号は1回だけ入れます。
    文書ごとに単語を `set` にしてから追加すると重複を避けられます。

??? tip "ヒント4"

    最後に通常の辞書へ変換します。
    出力例を安定させるには、単語をソートしながら新しい辞書を作ります。
