# 044: 「64文字の手紙」のヒント

[問題](../problems/044-base64-family.md) / [解答](../solutions/044-base64-family.md)

**難易度:** ☆☆☆☆☆

??? tip "ヒント1"

    3バイトは24ビットです。
    Base64では、この24ビットを6ビットずつ4つに分けます。
    6ビットで表せる値は0から63までなので、64文字のアルファベットに対応します。

??? tip "ヒント2"

    3バイト未満の最後のかたまりは、右側に0のバイトを足して3バイトとして扱えます。
    ただし、足したバイトの数だけ、出力の末尾を `=` に置き換えます。
    1バイトだけ余った場合は `==`、2バイト余った場合は `=` になります。

??? tip "ヒント3"

    デコードでは、4文字を6ビットずつの値に戻して24ビットの整数にします。
    `=` は値0として計算し、最後に取り出すバイト数を減らします。

??? tip "ヒント4"

    `alphabet` から逆引き用の辞書を作ると、デコードが書きやすくなります。
    URL-safe Base64はアルファベットが違うだけなので、標準版と同じ処理で扱えます。
