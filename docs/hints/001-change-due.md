# 001: 「釣りはいらねえ、とは言えなくて」のヒント

[問題](../problems/001-change-due.md) / [解答](../solutions/001-change-due.md)

**難易度:** ☆

??? tip "ヒント1"

    おつりは、支払額から合計金額を引いた値です。

??? tip "ヒント2"

    `paid - total` の形で計算できます。

??? tip "ヒント3"

    この問題では、`paid` が `total` 以上であることは入力の条件として与えられています。
