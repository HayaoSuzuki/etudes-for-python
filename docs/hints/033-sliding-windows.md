# 033: 「窓は一歩ずつ流れていく」のヒント

[問題](../problems/033-sliding-windows.md) / [解答](../solutions/033-sliding-windows.md)

**難易度:** ☆☆☆☆

??? tip "ヒント1"

    `iter(iterable)` でイテレータを作ると、リストやジェネレータを同じように扱えます。

??? tip "ヒント2"

    最初に `size` 個の要素を集められなければ、何も生成せずに終わります。

??? tip "ヒント3"

    新しい要素が来るたびに、古い要素を一つ捨てて新しいタプルを作ります。
    `collections.deque` の `maxlen` が使えます。

??? tip "ヒント4"

    ジェネレータ関数では、値を返す代わりに `yield` で順に値を出します。
