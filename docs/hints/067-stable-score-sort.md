# 067: 「同点の列は昨日を覚えている」のヒント

[問題](../problems/067-stable-score-sort.md) / [解答](../solutions/067-stable-score-sort.md)

**難易度:** ☆☆☆

??? tip "ヒント1"

    `sorted(entries, key=...)` は、元のリストを変更せずに新しいリストを返します。

??? tip "ヒント2"

    複数の条件で並べるには、キー関数でタプルを返します。
    タプルは左の要素から順に比較されます。

??? tip "ヒント3"

    高い得点を先にしたい場合は、キーに `-score` を使えます。

??? tip "ヒント4"

    Python のソートは安定です。
    キーが等しい要素は、入力での順序を保ちます。
