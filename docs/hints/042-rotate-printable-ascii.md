# 042: 「ASCIIは二度回る」のヒント

[問題](../problems/042-rotate-printable-ascii.md) / [解答](../solutions/042-rotate-printable-ascii.md)

**難易度:** ☆☆☆

??? tip "ヒント1"

    `!` から `~` までを回転対象にします。
    `ord("!")` は33、`ord("~")` は126です。

??? tip "ヒント2"

    対象範囲には94文字あります。
    文字コードから33を引くと、対象文字を0から93までの番号として扱えます。

??? tip "ヒント3"

    回転後の番号は、`(番号 + shift) % 94` で求められます。
    Pythonの `%` は、負の `shift` に対しても0以上の余りを返します。

??? tip "ヒント4"

    `shift` を47に固定したものが、前問のROT47です。
    固定値を関数の外から渡せるようにすると、ROT47以外の回転にも使えます。
