# 019: 「紅美鈴はよく眠る」のヒント

[問題](../problems/019-password-validator.md) / [解答](../solutions/019-password-validator.md)

**難易度:** ☆☆☆

??? tip "ヒント1"

    長さの検査は、正規表現ではなく `len(password)` で書くと単純です。
    文字の種類を調べるところで正規表現を使います。

??? tip "ヒント2"

    使える文字だけで構成されているかは、`re.fullmatch` で調べます。
    一部に含まれるかを調べたいときは、`re.search` を使います。

??? tip "ヒント3"

    小文字、大文字、数字、記号は、それぞれ別の正規表現で調べると読みやすくなります。
    たとえば小文字なら `[a-z]`、数字なら `[0-9]` です。

??? tip "ヒント4"

    同じ文字が3回連続するかは、後方参照で調べられます。
    `(.)\1\1` は、任意の1文字と同じ文字が続く形に一致します。
