# 079: 「点数表に割り込む」のヒント

[問題](../problems/079-bisect-score-board.md) / [解答](../solutions/079-bisect-score-board.md)

**難易度:** ☆☆☆

??? tip "ヒント1"

    `bisect` は、すでに昇順に並んだ列に対して、値を入れる位置を探します。

??? tip "ヒント2"

    点数表は高い点数ほど前にあります。
    昇順として扱うために、点数にマイナスを付けたキーのリストを作ると扱いやすくなります。

??? tip "ヒント3"

    同点の後ろへ入れるには、左端ではなく右端の挿入位置を探します。
    `bisect_right` が使えます。

??? tip "ヒント4"

    元のリストを変更しないために、スライスで前半と後半を作り、その間に新しいタプルを挟みます。
