# 020: 「名前順では勝てない表彰台」のヒント

[問題](../problems/020-score-podium.md) / [解答](../solutions/020-score-podium.md)

**難易度:** ☆☆☆

??? tip "ヒント1"

    `sorted()` は元のリストを変更せず、並べ替えた新しいリストを返します。

??? tip "ヒント2"

    点数は大きい順、名前は小さい順です。点数だけ符号を反転させると、1つのキーで表せます。

??? tip "ヒント3"

    `key=lambda record: (-record[1], record[0])` のように、並べ替え用の値を作れます。
