# 038: 「七桁の住所不定」のヒント

[問題](../problems/038-postal-code.md) / [解答](../solutions/038-postal-code.md)

**難易度:** ☆☆

??? tip "ヒント1"

    まず、空白とハイフンを除いた文字列を作ります。

??? tip "ヒント2"

    `for ch in text:` で1文字ずつ見て、残す文字だけを新しい文字列に足せます。

??? tip "ヒント3"

    長さが7で、すべて数字なら、先頭3文字と残り4文字を `-` でつなげます。
