# 014: 「零時をまたぐ分針」のヒント

[問題](../problems/014-minutes-until.md) / [解答](../solutions/014-minutes-until.md)

**難易度:** ☆☆☆

??? tip "ヒント1"

    まず、時刻を「0時からの分数」に変換すると計算しやすくなります。

??? tip "ヒント2"

    `split(":")` で時と分に分け、`int()` で整数に変換します。

??? tip "ヒント3"

    終了時刻の分数から開始時刻の分数を引き、負になったときだけ24時間ぶんを足します。
