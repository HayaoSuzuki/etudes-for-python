# 078: 「週末を数えない日付」のヒント

[問題](../problems/078-business-days.md) / [解答](../solutions/078-business-days.md)

**難易度:** ☆☆☆☆

??? tip "ヒント1"

    `date.fromisoformat("2026-06-30")` は、ISO形式の文字列から `date` オブジェクトを作ります。
    文字列の比較ではなく、日付型で扱います。

??? tip "ヒント2"

    1日進めるには `timedelta(days=1)` を足します。
    `date.weekday()` は月曜日を0、日曜日を6として返します。

??? tip "ヒント3"

    土日は `weekday() >= 5` で判定できます。
    祝日は、先に `date` の集合に変換しておくと判定しやすくなります。

??? tip "ヒント4"

    開始日は数えないので、ループの中で先に1日進めます。
    進めた日が営業日なら、残りの営業日数を1つ減らします。
