# 007: 「驚くべき関数」のヒント

[問題](../problems/007-even-or-odd.md) / [解答](../solutions/007-even-or-odd.md)

**難易度:** ☆

??? tip "ヒント1"

    偶数は、2で割った余りが0になる整数です。
    奇数は、その反対側にあります。

??? tip "ヒント2"

    Pythonでは、剰余を `%` で求めます。
    ただし、奇数を「余りが1」とだけ表すと、負の整数で剰余の規則を気にする必要が出ます。

    ```python
    >>> -3 % 2
    1
    ```

??? tip "ヒント3"

    奇数を「2で割り切れない整数」と考えると、比較する相手は1ではありません。
    0と比較する条件に言い換えられます。
