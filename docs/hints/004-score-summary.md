# 004: 「足して割れば、だいたい人生」のヒント

[問題](../problems/004-score-summary.md) / [解答](../solutions/004-score-summary.md)

**難易度:** ☆

??? tip "ヒント1"

    空のリストでは、平均を計算するために割り算をしてはいけません。

??? tip "ヒント2"

    合計は `sum(scores)`、個数は `len(scores)` で求められます。

??? tip "ヒント3"

    空でない場合だけ、`total / len(scores)` で平均を求めます。
