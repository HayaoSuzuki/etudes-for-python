# 035: 「メリー・ポピンズからの暗号文」のヒント

[問題](../problems/035-rot47-cipher.md) / [解答](../solutions/035-rot47-cipher.md)

**難易度:** ☆☆

??? tip "ヒント1"

    ROT47は、`!` から `~` までの印字可能ASCII文字を対象にします。
    `ord("!")` は33、`ord("~")` は126です。

??? tip "ヒント2"

    対象範囲には94文字あります。
    文字コードを33から始まる番号に直し、47を足して94で割った余りを使います。

??? tip "ヒント3"

    ROT47は、同じ変換を2回かけると元に戻ります。
    そのため、暗号化用と復号用の関数を分ける必要はありません。

??? tip "ヒント4"

    `chr()` と `ord()` を使うと、文字と文字コードを行き来できます。
