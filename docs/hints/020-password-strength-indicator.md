# 020: 「私の戦闘力は53万」のヒント

[問題](../problems/020-password-strength-indicator.md) / [解答](../solutions/020-password-strength-indicator.md)

??? tip "ヒント1"

    使える文字だけで構成されているかは、前問と同じように `fullmatch` で調べます。
    ここで失敗したら、点数を計算せず `"weak"` を返せます。

??? tip "ヒント2"

    小文字、大文字、数字、記号の有無は、`search` で調べます。
    条件と正規表現をペアにしてループすると、似た処理を短く書けます。

??? tip "ヒント3"

    同じ文字の3連続は `(.)\1\1` で調べられます。
    危険な単語の検出では、`re.IGNORECASE` を使うと大文字小文字を区別せずに調べられます。

??? tip "ヒント4"

    点数を先に計算し、最後に `if` で `"weak"`、`"medium"`、`"strong"` へ変換します。
    点数が負になっても、そのまま分類してかまいません。
