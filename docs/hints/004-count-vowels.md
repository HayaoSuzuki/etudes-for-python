# 004: 「AEIOUだけが知っている」のヒント

[問題](../problems/004-count-vowels.md) / [解答](../solutions/004-count-vowels.md)

**難易度:** ☆

??? tip "ヒント1"

    `for` 文で文字列を1文字ずつ取り出せます。

??? tip "ヒント2"

    大文字と小文字を同じに扱うには、文字を小文字にしてから判定します。

??? tip "ヒント3"

    `if ch in "aeiou":` のように書くと、母音かどうかを判定できます。
