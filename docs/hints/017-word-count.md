# 017: 「ことばたちの出席簿」のヒント

[問題](../problems/017-word-count.md) / [解答](../solutions/017-word-count.md)

**難易度:** ☆☆

??? tip "ヒント1"

    辞書では、単語をキー、出現回数を値として持てます。

??? tip "ヒント2"

    まだ辞書にない単語を見つけたら、回数を1にします。

??? tip "ヒント3"

    すでに辞書にある単語を見つけたら、今の回数に1を足します。
