# 010: 「合言葉はレモン」のヒント

[問題](../problems/010-vigenere-cipher.md) / [解答](../solutions/010-vigenere-cipher.md)

??? tip "ヒント1"

    英大文字は26文字の輪として扱えます。
    `ord("A")` を引くと、`A` から `Z` を0から25までの番号にできます。

??? tip "ヒント2"

    鍵の文字も同じように番号にします。
    たとえば `key` の文字が `L` なら、シフト量は `ord("L") - ord("A")` です。

??? tip "ヒント3"

    `text` の添字をそのまま鍵の添字に使うと、空白や記号でずれます。
    変換した英大文字の数を別に数え、その数を使って鍵の位置を決めます。

??? tip "ヒント4"

    暗号化はシフト量を足します。
    復号は同じシフト量を引きます。
