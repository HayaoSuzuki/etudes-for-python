# 044: 「一度だけ現れる声」のヒント

[問題](../problems/044-first-unique-character.md) / [解答](../solutions/044-first-unique-character.md)

**難易度:** ☆☆☆

??? tip "ヒント1"

    まず、文字ごとの出現回数を辞書に数えます。

??? tip "ヒント2"

    次に、元の文字列をもう一度先頭から見ます。

??? tip "ヒント3"

    出現回数が1の文字を見つけたら、その文字を返します。
