# 019: 「読書ログは期限を覚えている」のヒント

[問題](../problems/019-group-books-by-status.md) / [解答](../solutions/019-group-books-by-status.md)

**難易度:** ☆☆☆

??? tip "ヒント1"

    状態ごとのタイトル一覧は、辞書の値としてリストを持つと表せます。
    初めて見る状態では、空リストと冊数0を用意します。

??? tip "ヒント2"

    `YYYY-MM-DD` 形式の日付は、文字列のまま比較しても年月日の順になります。
    `"2026-06-20" < "2026-06-30"` は真です。

??? tip "ヒント3"

    期限切れの本は、あとで期限順に並べるために、最初は `(due, index, title)` の形で集めると扱いやすくなります。
    `index` は入力順を保つための番号です。

??? tip "ヒント4"

    最後に期限切れ候補を `sorted` で並べ、タイトルだけを取り出します。
    タプルは先頭の要素から順に比較されます。
