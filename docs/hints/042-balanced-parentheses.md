# 042: 「括弧の門番は眠らない」のヒント

[問題](../problems/042-balanced-parentheses.md) / [解答](../solutions/042-balanced-parentheses.md)

**難易度:** ☆☆☆

??? tip "ヒント1"

    開き括弧を見つけたら、まだ閉じていない括弧の数を1増やします。

??? tip "ヒント2"

    閉じ括弧を見つけたら、その数を1減らします。途中で負になったら、対応する開き括弧がありません。

??? tip "ヒント3"

    最後に、まだ閉じていない括弧の数が0なら正しく対応しています。
