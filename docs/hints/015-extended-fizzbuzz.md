# 015: 「七五三」のヒント

[問題](../problems/015-extended-fizzbuzz.md) / [解答](../solutions/015-extended-fizzbuzz.md)

**難易度:** ☆☆

??? tip "ヒント1"

    1から `n` までの整数は、`range(1, n + 1)` で順に取り出せます。

??? tip "ヒント2"

    倍数かどうかは、剰余で判定できます。
    たとえば、`i % 3 == 0` なら `i` は3の倍数です。

??? tip "ヒント3"

    15は3の倍数でもあり、5の倍数でもあります。
    このような数では、条件を1つ選ぶのではなく、当てはまる文字列を順に足します。

??? tip "ヒント4"

    最初に空文字列を用意し、3、5、7の条件に当たるたびに文字列を足すと書きやすくなります。
    最後まで空文字列のままなら、数値を文字列にします。
