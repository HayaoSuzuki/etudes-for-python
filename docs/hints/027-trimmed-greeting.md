# 027: 「空白を脱いだ名前」のヒント

[問題](../problems/027-trimmed-greeting.md) / [解答](../solutions/027-trimmed-greeting.md)

**難易度:** ☆

??? tip "ヒント1"

    文字列の前後の空白は、文字列メソッドで取り除けます。

??? tip "ヒント2"

    `name.strip()` を使うと、前後の空白を取り除いた新しい文字列が得られます。

??? tip "ヒント3"

    空文字列かどうかを `if name == "":` で確認してから、あいさつ文を組み立てます。
