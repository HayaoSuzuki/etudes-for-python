# 039: 「昨日と同じリストではない」のヒント

[問題](../problems/039-independent-board.md) / [解答](../solutions/039-independent-board.md)

**難易度:** ☆☆☆

??? tip "ヒント1"

    外側のリストに、行を1つずつ追加していきます。

??? tip "ヒント2"

    各行を作るたびに、新しい空リストから始めます。

??? tip "ヒント3"

    同じ行リストを何度も追加すると、行どうしが同じものを指してしまいます。
