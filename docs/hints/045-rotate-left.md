# 045: 「スライスが回す輪」のヒント

[問題](../problems/045-rotate-left.md) / [解答](../solutions/045-rotate-left.md)

**難易度:** ☆☆

??? tip "ヒント1"

    空リストでは、長さで割った余りを計算できません。先に特別扱いします。

??? tip "ヒント2"

    `n % len(items)` を使うと、何周しても同じ位置にできます。負の `n` にも対応できます。

??? tip "ヒント3"

    左に回す数を `n` とすると、`items[n:] + items[:n]` で回転できます。
