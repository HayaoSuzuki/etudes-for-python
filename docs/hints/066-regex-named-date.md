# 066: 「括弧に名前をつけた日」のヒント

[問題](../problems/066-regex-named-date.md) / [解答](../solutions/066-regex-named-date.md)

**難易度:** ☆☆☆

??? tip "ヒント1"

    `re.finditer` を使うと、見つかった箇所を順に `Match` オブジェクトとして扱えます。

??? tip "ヒント2"

    名前付きグループは `(?P<year>...)` のように書きます。
    取り出すときは `match.group("year")` または `match["year"]` が使えます。

??? tip "ヒント3"

    月は `0[1-9]|1[0-2]`、日は `0[1-9]|[12]\d|3[01]` のように範囲を分けると書きやすくなります。

??? tip "ヒント4"

    前後に数字が続くものを避けるには、直前と直後に数字がないことを正規表現で表します。
