# 085: 「入れ子の状態は奥まで写す」のヒント

[問題](../problems/085-deepcopy-nested-state.md) / [解答](../solutions/085-deepcopy-nested-state.md)

**難易度:** ☆☆☆

??? tip "ヒント1"

    辞書の `copy()` は、外側の辞書だけをコピーします。
    内側にあるリストや辞書は、元の状態と共有されたままになります。

??? tip "ヒント2"

    入れ子の値までまとめてコピーしたい場合は、`copy.deepcopy` を使います。
    コピーした状態を変更すれば、元の状態は変わりません。

??? tip "ヒント3"

    カードの移動は、`source` 列の先頭から取り出し、`target` 列の末尾へ追加します。
    先頭から取り出すには `pop(0)` が使えます。

??? tip "ヒント4"

    履歴文字列は、取り出したカード名がわかってから作ります。
    `f"{source}->{target}: {card}"` の形で `history` に追加します。
