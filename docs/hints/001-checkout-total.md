# 001: 「レジは明細を覚えている」のヒント

[問題](../problems/001-checkout-total.md) / [解答](../solutions/001-checkout-total.md)

**難易度:** ☆
??? tip "ヒント1"

    `cart` を1行ずつ処理しながら、商品コードごとの数量を辞書に足していきます。
    まだ辞書にない商品コードなら、先に0を入れてから足すと扱いやすくなります。

??? tip "ヒント2"

    明細の順番を保つには、商品コードを初めて見たタイミングを覚える必要があります。
    数量用の辞書とは別に、初登場順のリストを持つと素直に書けます。

??? tip "ヒント3"

    小計は明細を作るときに同時に足せます。
    税額、合計、おつりは、小計が決まったあとで順に計算します。

??? tip "ヒント4"

    支払額の不足は、合計を計算したあとで判定します。
    未登録の商品コードは、商品表から単価を取り出す前に見つけると、どこで失敗したかがはっきりします。
