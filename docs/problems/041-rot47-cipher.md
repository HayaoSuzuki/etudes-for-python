# 041: メリー・ポピンズからの暗号文

[ヒント](../hints/041-rot47-cipher.md) / [解答](../solutions/041-rot47-cipher.md)

**難易度:** ☆☆

## 問題

次の暗号文は、ROT47で暗号化されています。
復号する関数 `rot47` を書き、平文を求めてください。

```text
$FA6C42=:7C28:=:DE:46IA:2=:5@4:@FD
```

## 制約

- 入力は文字列です。
- `!` から `~` までのASCII文字をROT47の対象にします。
- 対象外の文字が含まれている場合は、そのまま残します。

## 例

```python
>>> rot47("Hello")
'w6==@'
>>> rot47("w6==@")
'Hello'

```
