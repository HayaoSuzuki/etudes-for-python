# 004: 足して割れば、だいたい人生

[ヒント](../hints/004-score-summary.md) / [解答](../solutions/004-score-summary.md)

**難易度:** ☆

## 問題

関数 `score_summary(scores)` を書いてください。
この関数は、点数のリスト `scores` を受け取り、合計点と平均点を `(合計, 平均)` のタプルで返します。

リストが空の場合は `(0, 0.0)` を返してください。

## 制約

- `scores` は整数のリストです。
- 各点数は0以上100以下です。
- 平均は `float` で返します。

## 例

```python
>>> score_summary([80, 90, 70])
(240, 80.0)
>>> score_summary([100])
(100, 100.0)
>>> score_summary([])
(0, 0.0)
```

## 発展

平均点を小数第1位まで丸める版も書いてください。
