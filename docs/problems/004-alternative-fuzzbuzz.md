# 004: 宇宙の鍵は3,6,9

[ヒント](../hints/004-alternative-fuzzbuzz.md) / [解答](../solutions/004-alternative-fuzzbuzz.md)

## 問題

整数 `n` を受け取り、1から `n` までの各整数を次の規則で文字列に変換する関数 `alternative_fuzzbuzz` を書いてください。

- 9の倍数なら `Buzz` にします。
- 6の倍数なら `Fuzz` にします。
- 3の倍数なら `Fizz` にします。
- 複数の条件を満たす場合は、上に書いた規則を優先します。
- どの条件も満たさない場合は、数値を文字列にします。

関数は、変換した文字列を順番に並べたリストを返します。

## 制約

- `n` は1以上の整数です。

## 例

```python
>>> alternative_fuzzbuzz(9) == [
...     "1",
...     "2",
...     "Fizz",
...     "4",
...     "5",
...     "Fuzz",
...     "7",
...     "8",
...     "Buzz",
... ]
True
>>> alternative_fuzzbuzz(18)[-1]
'Buzz'

```

## 発展

9、6、3の判定順を入れ替えると、18の結果がどう変わるかを確認してください。
