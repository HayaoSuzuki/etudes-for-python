# 027: 空白を脱いだ名前

[ヒント](../hints/027-trimmed-greeting.md) / [解答](../solutions/027-trimmed-greeting.md)

**難易度:** ☆

## 問題

関数 `greet(name)` を書いてください。
この関数は、名前を表す文字列 `name` を受け取り、`こんにちは、名前さん` という文字列を返します。

名前の前後に空白がある場合は取り除いてください。
空白を取り除いたあとに空文字列になる場合は、名前を `ゲスト` としてください。

## 制約

- `name` は文字列です。
- 前後の空白だけを取り除き、名前の途中にある空白は残します。

## 例

```python
>>> greet("Alice")
'こんにちは、Aliceさん'
>>> greet("  Python  ")
'こんにちは、Pythonさん'
>>> greet("山田 太郎")
'こんにちは、山田 太郎さん'
>>> greet("   ")
'こんにちは、ゲストさん'
```

## 発展

英字の名前だけを対象にして、先頭だけ大文字にそろえる版も考えてください。
