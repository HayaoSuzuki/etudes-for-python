# 030: リストの両端に王がいる

[ヒント](../hints/030-min-max.md) / [解答](../solutions/030-min-max.md)

**難易度:** ☆☆

## 問題

関数 `min_max(numbers)` を書いてください。
この関数は、数値のリスト `numbers` を受け取り、最小値と最大値を `(最小値, 最大値)` のタプルで返します。

この問題では、組み込み関数 `min()` と `max()` は使わず、`for` 文でリストを調べてください。

## 制約

- `numbers` は整数のリストです。
- `numbers` には少なくとも1個の要素があります。
- 要素には負の整数も含まれます。

## 例

```python
>>> min_max([3, 1, 4, 1, 5])
(1, 5)
>>> min_max([-2, -8, 0])
(-8, 0)
>>> min_max([7])
(7, 7)
```

## 発展

空のリストが渡されたら `ValueError` を送出する版も書いてください。
