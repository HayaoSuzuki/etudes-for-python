# 006: 四角い三角関係

[ヒント](../hints/006-square-triangular-number.md) / [解答](../solutions/006-square-triangular-number.md)

## 問題

**平方数**とは、ある正の整数 `m` を使って `m * m` と表せる数です。
たとえば、`1`、`4`、`9`、`16` は平方数です。

**三角数**とは、ある正の整数 `k` を使って `k * (k + 1) // 2` と表せる数です。
たとえば、`1`、`3`、`6`、`10` は三角数です。

平方数でも三角数でもある正の整数を、**平方三角数**と呼びます。
平方三角数を小さい順に10個返す関数 `square_triangular_numbers` を書いてください。

## 制約

- 引数 `count` は1以上10以下の整数です。
- 関数は、小さい順に並んだ平方三角数のリストを返します。

## 例

```python
>>> square_triangular_numbers(1)
[1]
>>> square_triangular_numbers(3)
[1, 36, 1225]

```

## 目標

```python
>>> square_triangular_numbers(10) == [
```
