# 023: スライスが回す輪

[ヒント](../hints/023-rotate-left.md) / [解答](../solutions/023-rotate-left.md)

**難易度:** ☆☆

## 問題

関数 `rotate_left(items, n)` を書いてください。
この関数は、リスト `items` を左に `n` 個ぶん回転した新しいリストを返します。

元のリストは変更しません。
`n` がリストの長さ以上の場合は、何周ぶんかを考えて同じ結果になるようにしてください。
`n` が負の場合は、右に回転するものとして扱います。

## 制約

- `items` はリストです。
- `n` は整数です。
- `items` が空の場合は、空のリストを返します。

## 例

```python
>>> rotate_left([1, 2, 3, 4], 1)
[2, 3, 4, 1]
>>> rotate_left([1, 2, 3, 4], 5)
[2, 3, 4, 1]
>>> rotate_left([1, 2, 3, 4], -1)
[4, 1, 2, 3]
>>> rotate_left([], 3)
[]
```

## 発展

文字列にも同じ考え方を適用して、文字列を回転する関数を書いてください。
