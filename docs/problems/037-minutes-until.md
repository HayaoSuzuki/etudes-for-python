# 037: 零時をまたぐ分針

[ヒント](../hints/037-minutes-until.md) / [解答](../solutions/037-minutes-until.md)

**難易度:** ☆☆☆

## 問題

関数 `minutes_until(start, end)` を書いてください。
この関数は、開始時刻 `start` から終了時刻 `end` までの分数を返します。

時刻はどちらも `"HH:MM"` 形式の文字列です。
終了時刻が開始時刻より前に見える場合は、日付をまたいだものとして扱います。
開始時刻と終了時刻が同じ場合は、0分とします。

## 制約

- `start` と `end` は `"HH:MM"` 形式です。
- `HH` は `00` から `23`、`MM` は `00` から `59` です。
- 入力の形式チェックはしません。

## 例

```python
>>> minutes_until("09:15", "10:00")
45
>>> minutes_until("23:50", "00:10")
20
>>> minutes_until("09:00", "09:00")
0
```

## 発展

`"9:05"` のように、時が1桁の入力も受け付けるようにしてください。
