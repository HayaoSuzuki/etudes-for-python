# 032: 二で割り切れる者たち

[ヒント](../hints/032-even-numbers.md) / [解答](../solutions/032-even-numbers.md)

**難易度:** ☆☆

## 問題

関数 `even_numbers(numbers)` を書いてください。
この関数は、整数のリスト `numbers` を受け取り、偶数だけを元の順番で並べた新しいリストを返します。

## 制約

- `numbers` は整数のリストです。
- 負の整数、0、正の整数を含みます。
- 元のリストは変更しません。

## 例

```python
>>> even_numbers([1, 2, 3, 4, 5, 6])
[2, 4, 6]
>>> even_numbers([1, 3, 5])
[]
>>> even_numbers([-2, -1, 0, 1, 2])
[-2, 0, 2]
```

## 発展

同じ方針で、奇数だけを返す関数も書いてください。
