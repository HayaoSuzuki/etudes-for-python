# 034: カンマの向こうに点がある

[ヒント](../hints/034-comma-separated-scores.md) / [解答](../solutions/034-comma-separated-scores.md)

**難易度:** ☆☆

## 問題

関数 `parse_scores(line)` を書いてください。
この関数は、カンマ区切りの点数を表す文字列 `line` を受け取り、整数のリストを返します。

各点数の前後に空白がある場合は取り除いてください。
空文字列が渡された場合は、空のリストを返してください。

## 制約

- `line` は文字列です。
- 空でない項目は、整数として解釈できる文字列です。
- 空文字列以外では、空の項目は含まれません。

## 例

```python
>>> parse_scores("80,90,70")
[80, 90, 70]
>>> parse_scores(" 100, 95, 88 ")
[100, 95, 88]
>>> parse_scores("")
[]
```

## 発展

不正な項目があった場合に、その項目を無視する版も書いてください。
